# クイックスタート: カート管理

**作成日**: 2026-02-08
**ブランチ**: `002-cart-management`

## 前提条件

- カタログ機能（001-catalog-listing）が実装済みであること
- Node.js 18+ がインストール済みであること
- 依存パッケージがインストール済みであること（`pnpm install`）

## 主要シナリオ

### シナリオ 1: 商品をカートに追加

1. `/catalog` で商品一覧を表示
2. 商品カードをクリックして詳細ページへ遷移
3. 「カートに追加」ボタンをクリック
4. 成功メッセージが表示される
5. ヘッダーのカートアイコンの件数が1増加する

**検証ポイント**:
- POST `/api/cart/items` が200を返す
- `cart-updated` イベントが発火する
- 在庫切れ商品ではボタンが無効

### シナリオ 2: カート内容を確認

1. ヘッダーの「カート」リンクをクリック
2. カートページに商品一覧が表示される
3. 各商品の画像・名前・単価・数量・小計を確認
4. 合計セクションで商品合計・消費税・総合計を確認

**検証ポイント**:
- 消費税 = `Math.floor(商品合計 * 0.1)`
- 総合計 = 商品合計 + 消費税

### シナリオ 3: 数量変更

1. カートページで数量を変更
2. 小計と合計が即時更新される

**検証ポイント**:
- 1〜99の範囲のみ許可
- 在庫数を超える変更はエラー

### シナリオ 4: 商品削除

1. カートページで削除ボタンをクリック
2. 確認ダイアログが表示される
3. 「削除する」を選択
4. 商品が削除され、合計が再計算される

**検証ポイント**:
- 「キャンセル」では削除されない
- 最後の商品を削除すると空状態メッセージが表示

### シナリオ 5: カート永続化

1. カートに商品を追加
2. ブラウザをリロード
3. カートの内容が保持されている

**検証ポイント**:
- ページ遷移後もカート保持
- リロード後もカート保持

| 商品合計 | 消費税（10%、切り捨て） | 総合計 |

| ¥1,000 | ¥100 | ¥1,100 |
| ¥1,999 | ¥199 | ¥2,198 |
| ¥15,800 | ¥1,580 | ¥17,380 |
| ¥33 | ¥3 | ¥36 |
