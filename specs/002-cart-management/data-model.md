# データモデル: カート管理

**作成日**: 2026-02-08
**ブランチ**: `002-cart-management`

## エンティティ

### カート（Cart）

購入者ごとに1つ存在するカートエンティティ。

| フィールド | 型 | 説明 | 制約 |
|-----------|-----|------|------|
| id | 文字列（UUID） | カートの一意識別子 | 必須、自動生成 |
| userId | 文字列 | カート所有者のユーザーID | 必須 |
| items | CartItem[] | カートアイテムの配列 | 空配列可 |
| subtotal | 数値 | 商品合計金額（税抜） | 0以上、整数 |
| itemCount | 数値 | カート内の商品種類数 | 0以上、整数 |
| createdAt | 日時 | 作成日時 | 自動設定 |
| updatedAt | 日時 | 更新日時 | 自動更新 |

**導出フィールド**（表示時に計算）:
- 消費税: `Math.floor(subtotal * 0.1)` — 合計に10%を乗じ端数切り捨て
- 総合計: `subtotal + 消費税`

### カートアイテム（CartItem）

カート内の1商品を表すエンティティ。

| フィールド | 型 | 説明 | 制約 |
|-----------|-----|------|------|
| productId | 文字列（UUID） | 商品ID（カタログへの参照） | 必須 |
| productName | 文字列 | 商品名（スナップショット） | 必須 |
| price | 数値 | 単価（スナップショット） | 0以上、整数 |
| imageUrl | 文字列 | 商品画像URL（スナップショット） | 任意 |
| quantity | 数値 | 数量 | 1〜99、在庫数以下 |
| addedAt | 日時 | カート追加日時 | 自動設定 |

**導出フィールド**:
- 小計: `price * quantity`

### 商品（Product）— 参照先

カタログ機能（001-catalog-listing）で管理される既存エンティティ。カートからは `productId` で参照する。

| 参照フィールド | 用途 |
|--------------|------|
| id | カートアイテムとの紐付け |
| name | カートアイテムの productName に使用 |
| price | カートアイテムの price に使用 |
| imageUrl | カートアイテムの imageUrl に使用 |
| stock | 数量上限チェック、在庫切れ判定 |

## リレーションシップ

```text
ユーザー 1 --- 1 カート（userId で紐付け）
カート 1 --- * カートアイテム（items 配列）
カートアイテム * --- 1 商品（productId で参照）
```

## バリデーションルール

1. **数量**: 1以上99以下の整数。在庫数（product.stock）を超えてはならない
2. **同一商品の重複**: カート内に同一 productId のアイテムは1つのみ。追加時は既存アイテムの数量を加算
3. **小計・合計**: 数量変更・アイテム追加・削除時に自動再計算
4. **消費税**: 全アイテムの小計合計に10%を乗じ、`Math.floor()` で端数切り捨て
